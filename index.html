¿Qué es el DOM?


El Document Object Model (DOM) es una interfaz de programación para documentos HTML y XML. Representa la estructura del documento como una jerarquía de nodos y objetos, permitiendo a los lenguajes de programación manipular, modificar y acceder dinámicamente al contenido, estructura y estilo de una página web.

Características principales

  • Estructura Jerárquica Organiza los elementos en forma de árbol, donde cada nodo representa una parte del documento.

  • Interactividad Permite modificar el contenido y los estilos de los elementos en tiempo real mediante JavaScript.

  • Actualización Dinámica Facilita la actualización dinámica de la página sin recargarla completamente.

Objetivos y utilidad en el desarrollo web

  • Manipulación de contenido: Permite agregar, eliminar o modificar elementos y texto en la página.
  • Interactividad: Hace posible responder a eventos del usuario como clics, teclas o movimientos del mouse.
  • Acceso estructurado: Facilita la navegación y búsqueda de elementos dentro del documento.
  • Desarrollo de aplicaciones dinámicas: Es esencial para frameworks modernos y aplicaciones web interactivas.

Métodos DOM de uso común

El DOM no solo permite modificar el contenido y la estructura de una página, sino que también es fundamental para la accesibilidad, la optimización del rendimiento y la integración con APIs modernas del navegador. Gracias al DOM, los frameworks como React, Vue y Angular pueden crear interfaces de usuario reactivas y eficientes.


Métodos Descripción

obtenerElementoPorId(id)

 Selecciona un elemento por su ID.

obtenerElementosPorNombreDeClase(clase)

 Selecciona todos los elementos con una clase determinada.

querySelector(selector)

 Selecciona el primer elemento coincidente.

querySelectorAll(selector)

 Selecciona todos los elementos coincidentes.

createElement(etiqueta)

 Crea un nuevo elemento HTML.

appendChild(nodo)

 Agrega un nodo secundario a un elemento.

eliminar()

 Elimina un elemento del DOM.

addEventListener(evento, función)

 Adjunta un controlador de eventos a un elemento.
